# Kitchen-Recipes

A Flutter application designed to make cooking simple and enjoyable. With seamless state management using **Riverpod**, powerful backend services from **Firebase**, beautiful animations, and an intuitive UI/UX, Kitchen Recipes provides a delightful experience for discovering, saving, and sharing recipes.

---

## ✨ Features

* 🧑‍🍳 Discover a wide variety of recipes with step-by-step instructions
* ❤️ Save your favorite recipes for quick access
* 🔎 Search and filter recipes by category, ingredients, or name
* 🌐 Share recipes with friends and family
* 🗂 Organized categories and modern UI layout
* 🎨 Smooth animations for enhanced user experience
* 🔐 Firebase Authentication for secure user login
* ☁️ Firebase Firestore for real-time recipe storage and retrieval
* 🛠 State management using Riverpod

## 🧰 Technologies Used

* **Flutter** – Cross-platform app development
* **Dart** – Main programming language
* **Firebase** – Backend services (Auth, Firestore)
* **Riverpod** – State management
* **Flutter Hooks** – For cleaner and more reactive code
* **Lottie** – For animations
* **Cloud Firestore** – Realtime database

---

## 📐 Architecture

This app follows a **feature-first** architecture. Each feature (e.g., home, auth, recipe) contains its own models, views, providers, and logic to maintain separation of concerns and scalability.
